## 📖 Project Overview

This is my **personal portfolio website** built with Next.js. It showcases my projects, technical skills, and provides a way to get in touch with me. The site is optimized for performance with server-side rendering and a clean, modern design.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Next.js 16 | React Framework with SSR/SSG |
| React 19 | UI Components |
| TypeScript 5 | Type-Safe JavaScript |
| Tailwind CSS 4 | Utility-First Styling |
| Framer Motion | Animations & Transitions |
| Radix UI | Accessible UI Primitives |
| Three.js + React Three Fiber | 3D Graphics & Visuals |
| React Hook Form + Zod | Form Handling & Validation |
| Lucide React | Icon Library |
| next-themes | Dark / Light Mode |
| Geist | Font |

---

## ✨ Key Features

- 🎯 Hero section with personal intro & CTA
- 🌀 Smooth animations powered by **Framer Motion**
- 🧊 3D visuals using **Three.js + React Three Fiber**
- 🛠️ Skills & Projects showcase section
- 🌙 Dark / Light mode with **next-themes**
- 📬 Contact form with **React Hook Form + Zod** validation
- ⚡ Optimized performance with Next.js SSG
- 📱 Fully **responsive** on all screen sizes
- 🔍 SEO-friendly with Next.js metadata

## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sanoyon211/sa-noyon-portfolio.git

# 2. Navigate into the project folder
cd sa-noyon-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev

# 5. Open in browser
# Visit: http://localhost:3000
```
